# MAAP (Madonna Ability Alliance Portal)

MAAP is a web-based application that centralizes record-keeping and administrative
workflows for organizations supporting individuals with special needs including
students and adults. The portal stores client profiles, support plans, medical and
educational histories, contact details, staff assignments, case notes, and activity
logs to help teams coordinate care and maintain accurate, accessible records.

Key features:

- Centralized client and staff profiles with demographic, contact, and support data.
- Case notes, activity logs, and support-plan tracking to document progress.
- Search, filtering, and basic reporting tools to help staff find and summarize data.
- Import/export and conversion utilities for migrating existing records (see `conversion/`).
- A simple, responsive frontend for dashboards and profile management (see `Madonna/`).
