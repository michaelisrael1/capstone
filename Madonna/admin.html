Admin Overview

Use the tabs above to manage profile field options or import/export data.

The Edit Options tab lets you add new choices or retire existing ones for Sports, Risks, Elementary Schools, and Program Tags that appear on client profiles.

The Retired Options tab shows options that have been retired. Retired options are hidden from users but preserved in the system, and can be restored at any time.

The Import / Export tab lets authorized users export staff or patient records as a spreadsheet, or upload a file to replace all records.

Access

Only Directors, Head Coordinators, and Program Coordinators can access this page. Data import is restricted to Directors.

About Export

Export downloads the selected records from the database into your chosen file format. Staff exports staff records, Patients exports client records, and All exports both.

About Import

Uploading a file will replace all existing client and staff records with the contents of the spreadsheet. This action cannot be undone.
